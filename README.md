# AOS
AOS - это ос на **assembly**, пока-что не *доделанный*, но можете зайти в мой ТГ канал и писать мне коды

1. AOS нужно запускать как: ***qemu-system-x86_64 -drive format=raw,file=os.img,if=floppy***
